# Linux System Anomaly Monitor

This project monitors Linux system behavior by reading kernel-exposed data
from the /proc filesystem and helps identify abnormal system activity.

## Purpose
- Understand Linux internals
- Build a baseline of normal system behavior
- Detect anomalies in CPU, file usage, and process activity

## Scope
- Works only on the user's own system
- Defensive and educational purposes only

## Technologies
- Python
- Linux /proc filesystem
